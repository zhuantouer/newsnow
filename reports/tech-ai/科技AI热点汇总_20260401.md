### 2. Anthropic Claude Code 512,000 行源码泄露 🔥🔥🔥
**[来源：The Hacker News / The Verge / TechRadar / WSJ]**
- Claude Code v2.1.88 的 npm 包中包含了一个 source map 文件，意外暴露了完整 TypeScript 代码库
- 泄露规模：**1,893 个 TypeScript 文件、512,000+ 行代码、53+ 种内置工具、95+ 种斜杠命令**
- 被泄露的 GitHub 仓库 `claw-code` 已获得 **84,000+ Stars、82,000+ Forks**，Anthropic 已发出 **8,100+ 份 DMCA 删除通知**
- **未发布功能曝光**：
  - 🐾 **Tamagotchi 风格"宠物"**：坐在输入框旁对编码做出反应
  - 🌙 **"梦模式"(Dream Mode)**：允许 Claude 在后台持续思考、迭代想法
  - 🤖 **KAIROS 持久代理**：后台自动修 bug、运行任务、发送推送通知
  - 🕵️ **"卧底模式"**：对开源仓库进行隐身贡献，系统提示明确写着"不要暴露你的身份"
  - 🛡️ **反蒸馏防御**：检测到竞争对手抓取输出时，自动注入虚假工具定义来「毒化」训练数据

- **供应链风险**：npm 上已出现 typosquat 包（`audio-capture-napi`、`color-diff-napi` 等），针对试图编译泄露源码的开发者
- **Anthropic 声明**："这是由人为错误导致的发布打包问题，而非安全漏洞。没有客户数据暴露。"

> ⚠️ **安全警示**：2026年3月31日 UTC 00:21~03:29 期间通过 npm 安装/更新 Claude Code 的用户，可能下载了受 Axios 供应链攻击影响的特洛伊化版本，建议立即降级并轮换密钥。

## 📌 六、前沿论文精选（arXiv / alphaXiv / HuggingFace 本周热门）

### ⭐ 1. Natural-Language Agent Harnesses (NLAHs)
- **作者**: Linyue Pan, Lexiao Zou, Shuo Guo
- **热度**: alphaXiv 151📌 / 3,092 阅读
- **核心**: 将 AI Agent 的控制逻辑外部化并用自然语言执行，实现 47.2% 任务成功率，显著优于原生代码
- **意义**: 降低 Agent 编排门槛，非程序员也能定义复杂的 Agent 工作流

### ⭐ 2. Meta-Harness: 端到端模型框架优化
- **作者**: Yoonho Lee, Roshen Nair, Qizheng Zhang
- **热度**: alphaXiv 103📌 / 2,078 阅读
- **核心**: LLM 框架端到端优化系统，文本分类 +7.7 pt，数学推理 +4.7 pt
- **意义**: 自动优化 LLM 推理流程，减少人工调参

### ⭐ 3. OpenClaw-RL: 通过对话训练任意 Agent
- **作者**: Princeton AI Lab
- **热度**: HuggingFace 147⬆ / GitHub 4.52k⭐
- **核心**: 使用异步训练、PRM judges 和 hindsight-guided distillation 实现从多模态交互信号中学习策略
- **意义**: "Just talk to train" 范式，大幅降低 Agent 训练门槛

> 💡 **zishuai 关注点**：OpenClaw-RL 的强化学习框架对推荐系统的 Agent 化有直接参考价值，特别是其 reward shaping 和 hindsight distillation 机制。

### ⭐ 4. GEMS: Agent-Native 多模态生成
- **作者**: Zefeng He, Siyuan Huang, Xiaoye Qu
- **热度**: HuggingFace 63⬆ / alphaXiv 18📌
- **核心**: Agent 原生多模态生成框架，采用迭代优化循环、分层压缩内存和可扩展技能模块
- **意义**: 使轻量级模型（6B）在复杂任务上超越大型模型，具有极高的工程实用价值

### ⭐ 5. Seeing with You: 感知-推理协同进化 (PRCO)
- **作者**: Ziqi Miao, Haonan Jia, Lijun Li（上海 AI 实验室）
- **热度**: alphaXiv 25📌
- **核心**: 解耦多模态 LLM 的视觉感知和符号推理，准确率 +7 pt，感知错误 -39.2%
- **意义**: 解决多模态模型"看得到但理解不了"的核心瓶颈

### ⭐ 6. LongCat-Next: 离散原生多模态统一模型
- **作者**: Meituan LongCat Team（美团）
- **热度**: alphaXiv 24📌
- **核心**: 利用 DiNA 框架将视觉和音频转换为离散 token，在 MathVista 等视觉理解任务上表现出色
- **意义**: 中国团队在统一多模态模型领域的突破性工作

> 💡 **zishuai 关注点**：美团 LongCat-Next 的离散 token 方法对多模态推荐（如图文理解 + 推荐融合）有潜在启发。

### ⭐ 7. Think Anywhere in Code Generation
- **作者**: Xue Jiang, Tianyu Zhang, Ge Li
- **热度**: alphaXiv 11📌
- **核心**: 允许 LLM 在代码生成的任意 token 位置动态调用内部推理，pass@1 达 70.3%
- **意义**: "随时思考"机制可应用于任何需要动态决策的生成任务

### ⭐ 8. HyperP: 超球体优化重新审视语言模型缩放
- **作者**: Liliang Ren, Yang Liu, Yelong Shen（Microsoft）
- **热度**: alphaXiv 18📌
- **核心**: 利用可迁移的超球体优化建立 LLM 综合缩放定律，计算效率提高 **3.38×**
- **意义**: 为 LLM 训练和推理的资源优化提供理论框架

### ⭐ 9. CARLA-Air: 无人机 + 自动驾驶统一仿真
- **作者**: tianlezeng 等
- **热度**: HuggingFace 230⬆
- **核心**: 在 Unreal Engine 中整合高保真驾驶和多旋翼飞行仿真，支持空地联合 Agent 建模
- **意义**: Embodied AI 研究的重要基础设施

### ⭐ 10. Internal Safety Collapse in Frontier LLMs
- **作者**: 10 位研究者
- **热度**: HuggingFace 30⬆ / GitHub 831⭐

## 原文精读（含注释）

Anthropic on Tuesday confirmed that the internal code for its popular artificial intelligence (AI) coding assistant Claude Code was inadvertently `[inadvertently: 无意中，疏忽地——正式用语，常见于事故/安全报道]` published as a result of a human error.

"No sensitive customer data or credentials `[credentials: 凭据/凭证——安全领域专用，指 API key、密码、token 等]` were involved or exposed," an Anthropic spokesperson said in a statement shared with CNBC news. "This was a release packaging issue caused by human error, not a security breach `[security breach: 安全漏洞/安全事件——breach 在安全语境中指"突破防线"，与 vulnerability（漏洞）不同，breach 强调已实际发生]`. We are rolling out `[rolling out: 推出/部署——短语动词，科技新闻中极高频，指逐步发布新功能或修复]` steps to prevent this from happening again."

The discovery came after the AI startup pushed out `[pushed out: 推送/发布——开发语境下指将代码发布到仓库]` version 2.1.88 of the Claude Code npm package, with users noticing `[noticing: 注意到——这里用现在分词作伴随状语，表示"用户注意到的情况下"]` that the release included a source map file that could be used to access Claude Code's source code — comprising `[comprising: 包含/由……组成——注意 comprise 不能写成 "comprised of"，正式英语中直接用 "comprising"]` close to 2,000 TypeScript files and over 512,000 lines of code. The release is no longer available for download from npm.

Security researcher Chaofan Shou was the first to publicly flag `[flag: 标记/标出——动词用法，安全领域中指"发现并报告某个问题"]` the matter on X, stating "Claude code source has leaked through a map file in its npm registry!" The X post has since garnered `[garnered: 获得/赢得——比 "got" 更正式，常用于新闻报道中描述关注度]` over 28.8 million views. The leaked codebase continues to be accessible via a public GitHub repository that has amassed `[amassed: 积累/聚集——比 "collected" 更强调大量快速积聚]` over 84,000 stars and 82,000 forks.

The kind of source code leak is significant as it gives software developers, and Anthropic's rivals, a blueprint `[blueprint: 蓝图——比喻用法，指了解某个系统内部运作的"设计图"]` into how the popular coding tool works under the hood `[under the hood: 在引擎盖下/内部运作——常用科技惯用语，指软件或系统的内部实现]`. Users who dug into `[dug into: 深入研究——dig into 的过去式，科技报道中指深入分析代码或数据]` the code have posted details of its self-healing memory architecture `[self-healing memory architecture: 自愈记忆架构——AI 系统中指能自动修复和维护上下文记忆的设计]` to overcome the model's fixed context window limitations, alongside other internal components.

These include:

- A tool system for facilitating `[facilitating: 促进/实现——正式动词，指使某事变得容易或可能]` various functionalities, such as file reading or bash execution
- A query engine for handling LLM API calls and orchestration `[orchestration: 编排——AI/DevOps 领域术语，指协调多个组件或服务的执行流程]`
- Multi-agent orchestration for spawning `[spawning: 生成/派生——计算机术语，指创建新的进程或子 Agent]` "sub-agents" or swarms `[swarms: 蜂群——AI Agent 领域借用生物学概念，指大量协同工作的 Agent 集合]` to perform complex tasks
- A bidirectional communication layer `[bidirectional communication layer: 双向通信层——指两个组件之间可以互相发送消息的接口层]` that connects IDE extensions and the Claude Code CLI

The leak has also shed light on `[shed light on: 揭示/阐明——高频惯用语，比 "reveal" 更学术化]` a feature called KAIROS, which allows Claude Code to run as a persistent background agent `[persistent background agent: 持久后台代理——指不需要用户主动触发就能持续运行的 AI Agent]` that could periodically fix bugs or run tasks on its own without waiting for human input, and even send push notifications to the user. Complementing `[complementing: 补充/与...相辅相成——注意区分 complement（补充）和 compliment（赞美）]` this proactive mode is a new "dream" mode that would allow Claude to keep thinking in the background to generate ideas and iterate `[iterate: 迭代——技术领域核心动词，指重复改进]` on existing ones.

Perhaps the most intriguing `[intriguing: 引人入胜的/耐人寻味的——比 interesting 更强调好奇心]` detail is the tool's "undercover mode" `[undercover mode: 卧底模式——undercover 通常用于间谍/警察语境，这里指隐秘操作]` for making "stealth" contributions to open-source repositories. "You are operating UNDERCOVER in a PUBLIC/OPEN-SOURCE repository. Your commit messages, PR titles, and PR body MUST NOT contain any Anthropic internal information. Do not blow your cover `[blow your cover: 暴露身份——间谍/惊悚片常用短语]`," the system prompt reads.

Another fascinating `[fascinating: 引人注目的——比 interesting 程度更强]` finding concerns Anthropic's covert `[covert: 秘密的/隐蔽的——正式用语，常用于安全和情报语境]` attempt to fight back against model distillation attacks `[model distillation attacks: 模型蒸馏攻击——指竞争对手通过抓取模型输出来训练自己的模型]`. The system has controls in place that, should a rival attempt to scrape `[scrape: 抓取——数据采集领域术语，指自动提取网页或 API 数据]` Claude Code's outputs, inject bogus `[bogus: 虚假的/伪造的——口语化但在安全报道中也常用]` tool definitions into the API requests to poison `[poison: 投毒/污染——安全领域术语，指向数据或系统中注入恶意内容]` the training data.

### Typosquat npm Packages Pushed to Registry

With Claude Code's internals now fully exposed, the development risks arming bad actors `[bad actors: 恶意行为者——安全领域标准术语]` with ammunition `[ammunition: 弹药——比喻用法，指攻击者可利用的信息或工具]` to bypass guardrails `[guardrails: 防护栏——AI 领域指防止模型生成有害内容的安全机制]` and trick the system into performing unintended actions, such as running malicious commands or exfiltrating `[exfiltrating: 窃取/外泄——安全领域专用，指从系统中偷偷提取数据]` data.

"Rather than brute-forcing `[brute-forcing: 暴力破解——安全术语，指尝试所有可能组合来破解]` jailbreaks `[jailbreaks: 越狱——AI 安全领域指绕过模型的安全限制]` and prompt injections `[prompt injections: 提示注入——AI 安全核心漏洞类型，指通过恶意输入操控模型行为]`, attackers can now study and fuzz `[fuzz: 模糊测试——安全测试技术，向系统输入随机/畸形数据以发现漏洞]` exactly how data flows through Claude Code's four-stage context management pipeline `[pipeline: 流水线/管道——计算机领域指数据处理的多阶段流程]` and craft payloads `[payloads: 有效载荷/攻击负载——安全领域指恶意代码或数据包]` designed to survive compression, effectively backdooring `[backdooring: 植入后门——动词化用法，指在系统中安装隐蔽的未授权访问通道]` sessions of arbitrary `[arbitrary: 任意的——技术领域指不受限制的]` length," AI security firm Straiker said.

A more pressing concern is the fallout `[fallout: 后果/影响——原指核辐射尘降，现广泛用于指某事件的后续影响]` from the Axios supply chain attack, as users who installed or updated Claude Code via npm between March 31, 2026, 00:21 UTC and 03:29 UTC, may have also downloaded a trojanized `[trojanized: 被植入木马的——形容词化用法，指被恶意篡改的软件]` version of the HTTP client harboring `[harboring: 包含/藏匿——harboring 暗示隐蔽携带]` a cross-platform remote access trojan `[remote access trojan (RAT): 远程访问木马——允许攻击者远程控制受害者计算机的恶意软件]`. Users are recommended to immediately downgrade to safe versions and rotate all secrets `[rotate all secrets: 轮换所有密钥——安全最佳实践，指更换所有 API key、密码等]`.

Furthermore, attackers are capitalizing on `[capitalizing on: 利用——指利用某个机会或事件获益]` the leak to typosquat `[typosquat: 拼写抢注——安全攻击手法，注册与合法包名相似的恶意包名]` internal npm package names in an attempt to target those who may be trying to compile the leaked Claude Code source code and mount dependency confusion attacks `[dependency confusion attacks: 依赖混淆攻击——攻击者在公共注册表中发布与内部包同名的恶意包]`.

"They're currently empty stubs `[stubs: 存根——编程术语，指只有空壳没有实际功能的代码]` (`module.exports = {}`), but that's how these attacks work — squat `[squat: 抢占——typosquat 中的核心动词，指占用他人名称]` on the name, wait for downloads, then push a malicious update that hits everyone who installed it," security researcher Clément Dumas said in a post on X.

This is the second major gaffe `[gaffe: 失误/出丑——比 mistake 更强调公开场合的尴尬失误]` for Anthropic in the span of `[in the span of: 在……的时间范围内——正式时间表达]` a week. Last week, details about the company's upcoming AI model as well as other internal data were left accessible `[were left accessible: 被遗留为可访问状态——被动语态强调疏忽]` through the company's content management system (CMS). Anthropic subsequently acknowledged it had been testing the model with early access customers, and described it as its "most powerful model built to date `[to date: 迄今为止——正式英语常用时间表达]`."

### 技术写作模式

1. **安全事件报道的标准结构**：事件描述 → 官方声明引用 → 技术细节展开 → 影响分析 → 建议措施。本文严格遵循这一模式，值得模仿。

2. **被动语态的策略性使用**："No sensitive customer data or credentials **were** involved or exposed" — 安全声明中大量使用被动语态以避免直接承认责任方。

3. **比喻的精准运用**：`blueprint`（蓝图）、`ammunition`（弹药）、`poison`（投毒）— 安全报道善于用军事/工程比喻让技术概念更直观。

4. **引用的层次化处理**：先引用官方声明（"No sensitive data..."），再引用安全公司分析（Straiker），最后引用独立研究者（Clément Dumas），形成权威度递减但深度递增的引用链。

### 语法亮点

1. **独立主格结构**："Users **noticing** that the release included a source map file" — 现在分词作伴随状语，压缩信息密度。

2. **虚拟条件句**："should a rival attempt to scrape Claude Code's outputs" — 使用倒装的 should 替代 if，更正式更简洁，常见于法律和安全文本。

3. **破折号的信息补充**："comprising close to 2,000 TypeScript files and over 512,000 lines of code — **comprising**" 后面直接用破折号补充细节，是英文技术写作的高效手法。

4. **动词名词化**："The development risks **arming** bad actors with ammunition" — 动名词作主语补足语，让句子结构紧凑。
